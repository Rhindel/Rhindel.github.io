SPINE



The SPINE project (Structured Planning and Implementation of New Explorations) is one of the main project of the Center for Neurological Imaging, a research laboratory of the Brigham and Women's Hospital.


Concept

SPINE is a "virtual laboratory" that wishes to empower anyone willing to advance science. Backed up with a database of more than 20,000 brain MRIs, it is at the same time an educational platform for the general public and a place where scientists can conduct experiments relying on crowdsouring. Experts save precious time as segmenting brain structures is very time-consuming.

A first version of the website was implemented in 2013 and a game version was later developed.


Project

The SPINE website grew bigger than expected. As the original architecture of the code was not suitable for the addition of multiple plugins throughout the years, it became difficult for developers to implement new features and also slowed down the user experience.

After finishing my master thesis on "Exploration and annotation of medical data" at the CNI, I was offered to stay in order to start the implementation of a new SPINE. At first unfamiliar with web development, I started learning how to use javascript libraries including React.js and Redux. The website is currently in development.


The SPINE game was also pushed into a 3D environment by means of a plugin mirroring the screen of the computer onto a virtual screen in Unity3D.
